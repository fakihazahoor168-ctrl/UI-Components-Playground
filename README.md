# UI-Components-Playground
A web-based UI component builder that allows users to quickly create and customize HTML components with live code preview. Ideal for students, developers, and designers who want to experiment with Bootstrap-based components visually.
---

## **Features** 
- **Supported Components:**  
  - Buttons  
  - Cards  
  - Navbar (with editable brand and menu items)  
  - Modals  
  - Input fields (with editable placeholder)  
  - Alerts  
  - Badges  
  - Progress Bars  
  - Loaders / Spinners  
- **Live Customization Controls:** Modify text, colors, font sizes, border radius, shadows, and more.  
- **Live HTML Code Preview:** Automatically updates as you edit components.  
- **Copy HTML:** Copy the generated HTML with one click.  
- **Responsive Design:** Built using Bootstrap 5.

---

## **Tech Stack**

- **Frontend:** HTML, CSS (Bootstrap 5), JavaScript  
- **Drag & Drop:** Native HTML5 Drag & Drop API  
- **Fully Client-Side:** No backend required
